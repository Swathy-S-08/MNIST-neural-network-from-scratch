# MNIST Neural Network From Scratch

A neural network built entirely from scratch using NumPy without relying on machine learning frameworks such as TensorFlow or PyTorch.

## Features

* Dense (Fully Connected) Layers
* ReLU Activation
* Softmax Activation
* Categorical Cross-Entropy Loss
* Backpropagation
* Adam Optimizer
* Mini-Batch Training
* MNIST Dataset Classification
* Handwritten Digit Prediction
* Model Weight Saving and Loading

## Results

* Test Accuracy: 97.4% - 98.2%

## Requirements

* NumPy
* Matplotlib
* Pillow

## How to Run

### Train the Model

python mnist_classifier.py

### Predict a Handwritten Digit

python predict_handwritten_digit.py
